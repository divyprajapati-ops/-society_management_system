Application Walkthrough

1. Login Page

Professional login interface with role-based authentication. Users enter credentials and are redirected to their respective dashboards based on their role.

2. Admin Dashboard

Comprehensive admin dashboard with system overview, user management, building statistics, and quick access to all administrative functions.

3. Society Fund Management

Complete financial management system with fund tracking, contribution management, expense tracking, and detailed financial reports.

4. Buildings Management

Manage all society buildings with complete information, unit details, and resident assignments.

5. Users Management

Complete user management system with role assignments, profile management, and access control.

6. Pramukh Dashboard

Society Pramukh dashboard with overview of all buildings, residents, meetings, and maintenance activities.

7. Building Admin Dashboard

Building-specific dashboard for managing residents, maintenance, meetings, and building-specific operations.

8. Member Dashboard

Personal member dashboard showing profile information, maintenance requests, meeting schedules, and society announcements.

9. Maintenance Management

Complete maintenance tracking system with request management, vendor coordination, and repair scheduling.

10. Meeting Management

Schedule and manage society meetings with attendance tracking, agenda management, and minute recording.

Database Schema

Core Tables

Users Table

Stores user authentication and profile information with role-based access. 

users (id, name, email, password, role, building_id, created_at, last_login)

Buildings Table

Contains society building information and unit details. 

buildings (id, name, address, total_units, created_at)

Maintenance Table

Tracks maintenance requests and their status. 

maintenance (id, user_id, building_id, description, status, created_at)

API Endpoints
